Class Project Proposal

  • Product Name: Which FNAF Character Are You?
  • Team Members: Shannon Tierney
  • Application Overview:
    • This application will be an interactive personality quiz inspired by the video game Five Nights at Freddy's
    • Users will answer a series of personality-based questions
    • Based on a user's answers, they will be matched with an FNAF character
    • The final result will include a character name and description
    • The goal is to create an engaging and fun experience with spooky undertones
  • Technology Overview:
    • Languages: HTML, CSS, JavaScript
    • Platform: Web browser
    • Hosting: Microsoft?
    • Tools: VS Code
  • Proposal Last Updated: 3/29/26
  • Feature List:
    • Multiple personality-based questions
    • Answer buttons for user input
    • Score tracking system
    • Final character result display
    • Styled FNAF-theme
    • Restart quiz option
    • Progreoss indicator (i.e., 1/10, 2/10. etc)
  • Future Feature List:
    • Character results and expanded results
    • Personality matching system
    • Save/share results
    • Mobile-friendly design
    • Animations (maybe)
    • Dark mode toggle
  • Basic Technical Features:
    • Variables and data storage
    • Conditional statements (if/else)
    • Functions for quiz logic
    • Arrays/objects for questions and answers
    • Event handling (button clicks)
  • Advanced Technical Features:
    • Result calculation system
    • Randomized question order
    • State management for quiz progress
    • Responsive design
    • Local storage for saving results
    • CSS animations and transitions
